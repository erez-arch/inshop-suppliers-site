# INSHOP Suppliers — אתר תדמיתי

חלק מפרויקט **INSHOP SUPPLIERS**. אתר שיווקי בעברית שמציג את כל תכונות המערכת: קליטת סחורה עם OCR, אימות נאמן, דרישות זיכוי, תשלומים, הזמנות חכמות, פורטל ספק, המקרר החכם ועוד.

**אתר חי:** https://erez-arch.github.io/inshop-suppliers-site/

| קובץ | תיאור |
